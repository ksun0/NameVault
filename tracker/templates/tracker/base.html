Our Algorithm

In League of Legends, there are two ways to claim an account name. One is to create a new account with that name, and the other is to purchase a “Summoner Name Change” on an existing account. League uses a system to automatically release names from inactive accounts, based on the player’s level and length of inaction.

When you use NameVault to search for a League name, you will see you one four results:
label

Free

You will see this if no one in the listed region has this name. You can claim it through creating a new account or by purchasing a “Summoner Name Change” from an existing account.

label

Available

You will see this if a player in the listed region has this name, but has been inactive for so long that it can be taken from them. This can only be done through the use of a “Summoner Name Change.”

label

Taken

You will see this if an active player in the listed region has this name. If the player remains inactive from this day forward, the name will become available on the listed day.

label

Invalid Name

You will see this if you used a search term that uses a character League of Legends does not accept for display names. These vary by region.

{% elif 'Minecraft' in games %}

Our Algorithm

In Minecraft, there are two ways to claim an account name. One is to create a new account with that name, and the other is to perform a free name change on an existing account (this can only be done once per month). Minecraft does not automatically release names from inactive accounts, so there is no way for us to entirely predict when a name will become available.

When you use NameVault to search for a Minecraft name, you will see you one three results:
label

Free

You will see this if no Minecraft player currently has this name. In this case, you can claim this name with a new account, or with a free name change on an existing account.

label

Taken

You will see this if a player currently has this name. You will not be able to take this name unless that player chooses to change their name to something else.

label

Invalid Name

You will see this if you used a search term that uses a character Minecraft does not accept for display names. This list includes all characters except for English letters, numbers, and underscores.

{% elif 'RuneScape' in games %}

Our Algorithm

In RuneScape, there are two ways to claim an account name. One is to create a new account with that name, and the other is to perform a name change on an existing account. This process can be done once per month for free on members accounts, and additional swaps can be purchased with game bonds. RuneScape does not automatically release names from inactive accounts, so there is no way for us to entirely predict when a name will become available. However, RuneScape does occasionally clear names off a large number of inactive accounts, in which case they will all become available at once.

When you use NameVault to search for a RuneScape name, you will see you one three results:
label

Free

You will see this if no Runescape player currently has this name. In this case, you can claim this name with a new account, or with a name change on an existing account.

label

Taken

You will see this if a player currently has this name. You will not be able to take this name unless that player chooses to change their name to something else or the RuneScape mods perform a mass clear of inactive display names.

label

Invalid Name

You will see this if you used a search term that uses a character RuneScape does not accept for display names. This list includes all characters except for English letters, numbers, hyphens, and spaces.
